<h2>🧐 about project</h2>

<p>my application has filtering by name, category and price, as well as sorting by price, pagination</p>

<h2>🛠️ Installation Steps:</h2>

<p>1. Install npm modules</p>

```
npm install
```

<p>2. Start development server using below command</p>

```
npm dev
```

<p>3. Create production build using below command</p>

```
npm build
```

<p>4. Check production build locally using below command</p>

```
npm start
```
